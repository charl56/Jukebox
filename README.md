# Launch project

## Required : 
```
node
npm
python
pip
mysql
```

## Git : pull project
```
git clone https://github.com/charl56/Jukebox.git
```

## Frontend : 
```
cd frontend
npm install
npm run serve
```

## Backend :
```
cd backend
python -m venv env
(powerShell) env/Script/active
pip install -R requirements.txt
python app.py
```

## Database :
```
mysql -u {username} -p
{password}
'Run commands in init.sql'
```
